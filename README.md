# Bikeshare Project 

This project analyzes bikeshare data using Python. 
It allows users to explore usage statistics and system patterns across different cities.

## Information about how to use your project

### Project Setup Instructions:

Follow these steps to install and run the project:

1.Install Python 3 on your machine.
2.Clone the project repository:
`git clone https://github.com/your-username/post-your-work-project.git`

3.Navigate to the project folder:
`cd post-your-work-project`



### Dependencies:

You may need to install the following Python packages:

- pandas

- numpy

- time

- statistics

Install using:
`pip install pandas numpy`


### How to Use the Project:

Run this script:
`python3 bikeshare.py`

You will be asked to choose:

- City

- Month

- Day

Then the program will display:

- Most common travel times

- Popular stations

- Trip durations

- User statistics



### Common Issues / Troubleshooting:

- FileNotFoundError:
Make sure the `.csv` file is in the same folder and is NOT tracked (it should be listed in `.gitignore`).
- ModuleNotFoundError:
 Install the missing package using:
`pip install <package-name>`
